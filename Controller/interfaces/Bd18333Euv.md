## 公共方法

| 名称 | 参数 | 返回类型 | 描述 |

| AddDev | addr: string | void | 添加一个地址的芯片 |
| SetPwmDuty | addr: string, chIndex: string, pwm: string | void | 设置目标芯片的目标通道的PWM |
| SetPwmCurr | addr: string, chIndex: string, curr: string | void | 设置目标芯片的目标通道在PWM模式下的电流 |
| DimSet | addr: string, chIndex: string, dc: string | void | 设置目标芯片的目标通道的DC电流值 |
| ChannelOn | devAddr: string, chIndex: string | void | 打开目标芯片的目标通道 |
| ChannelOff | devAddr: string, chIndex: string | void | 关闭目标芯片的目标通道 |
| ChannelAllOn | devAddr: string | void | 打开目标芯片的所有通道 |
| ChannelAllOff | devAddr: string | void | 关闭目标芯片的所有通道 |
| ChannelOddOn | devAddr: string | void | 打开目标芯片的单数通道 |
| ChannelEvenOn | devAddr: string | void | 关闭目标芯片的双数通道 |

## 方法详细说明

### AddDev

**描述**: 添加一个地址的芯片

**参数**:
- `addr` (string): 设备地址

**示例代码**:
```csharp
controller.AddDev("5");
// 或使用4位二进制格式
controller.AddDev("0101");
```

---

### SetPwmDuty

**描述**: 设置目标芯片的目标通道的PWM

**参数**:
- `addr` (string): 设备地址
- `chIndex` (string): 通道索引 (0-24)
- `pwm` (string): PWM值 (0-100)

**示例代码**:
```csharp
controller.SetPwmDuty("5", "1", "50");
```

---

### SetPwmCurr

**描述**: 设置目标芯片的目标通道在PWM模式下的电流

**参数**:
- `addr` (string): 设备地址
- `chIndex` (string): 通道索引 (0-24)
- `curr` (string): 电流值

**示例代码**:
```csharp
controller.SetPwmCurr("5", "1", "128");
```

---

### DimSet

**描述**: 设置目标芯片的目标通道的DC电流值

**参数**:
- `addr` (string): 设备地址
- `chIndex` (string): 通道索引 (0-24)
- `dc` (string): DC电流值

**示例代码**:
```csharp
controller.DimSet("5", "1", "100");
```

---
