## 方法详细说明

### AddDev

**描述**: 添加一个地址的芯片

**参数**:
- `addr` (string): 地址

**示例代码**:
```csharp
controller.AddDev("0001");
```

---

### ReadStatus

**描述**: 读目标芯片的LED状态

**参数**:
- `addr` (string): 目标芯片地址

**示例代码**:
```csharp
controller.ReadStatus("0001");
string shortStatus = controller.Dev0Led1To24ShortErrorStatus;
string openStatus = controller.Dev0Led1To24OpenErrorStatus;
```

---

### SetPwmDuty

**描述**: 设置目标芯片的目标通道的PWM

**参数**:
- `addr` (string): 目标芯片地址
- `chIndex` (string): 通道索引 (1-24)
- `pwm` (string): PWM值 (0-100)

**示例代码**:
```csharp
controller.SetPwmDuty("0001", "1", "50");
```

---

### DimSet

**描述**: 设置目标芯片的目标通道的DC电流值

**参数**:
- `addr` (string): 目标芯片地址
- `chIndex` (string): 通道索引 (1-24)
- `dc` (string): DC电流值

**示例代码**:
```csharp
controller.DimSet("0001", "1", "3.75");
```

---

### ChannelOn

**描述**: 打开目标芯片的目标通道

**参数**:
- `devAddr` (string): 芯片地址
- `chIndex` (string): 通道索引

**示例代码**:
```csharp
controller.ChannelOn("0001", "1");
```

---

### ForeachReadErrors

**描述**: 循环读取Errors

**参数**:
- `devAddr` (string): 芯片地址
- `delayMs` (string): 延迟毫秒

**示例代码**:
```csharp

string openErrors = controller.ForeachReadOpenErrors;
```
